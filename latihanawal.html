Apa itu html

HTML itu adalah sebuah bahasa yang menggunakan markup atau penanda untuk membuat halaman web. Penanda atau markup ini, nanti akan kita sebut dengan HTML berperan untuk menentukan struktur dan tampilan dari sebuah web. Kalau kiata ibaratkan nih HTML Itu seperti batu untuk membangun rumah. Batu bata ini dapat disusun ,hingga menjadi fondasi dasar. HTML memiliki kelebihan seperti memiliki sumber memiliki summber komunitas yang cukup besar dengan pengunaan yang luas. Tidak hanya itu, kelebihan lainnya HTML mudah untuk dipelajari oleh seorang Web Developer pemula. Kelebihan lainnya adalah HTML merupakan bahasa pemrograman yang gratis atau open source
